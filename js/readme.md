## Building for Web Audio

Install the Emscripten SDK and Cheerp

```
cd [root]/js/purejs
make  
```

### Script Processor Node Build

```
cd [root]/js/script-processor-node
make
```

The library can be found in the `build` subfolder


### Audio Worklet Build

```
cd [root]/js/audio-worklet
make
```

The library can be found in the `build` subfolder
